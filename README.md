# mern-travel-agent

## Local Development


### .env
Add an .env file to the root of the project. You can copy and paste the 'Sample .env file' message into it if you please.

The two variables needed is MONGODB_URL and ACCESS_TOKEN_SECRET.

### Running
Then, open two terminal windows. On the root of the project, run `npm run start`. Then, in the frontend directory, run `npm run dev`.
